¿Qué es Vue.js?

Según su documentación oficial, Vue.js es un framework progresivo de JavaScript para construir interfaces de usuario, que puede adoptarse de forma incremental [1]. Fue creado por Evan You y lanzado públicamente en febrero de 2014 [2].

¿Por qué se eligió para este proyecto?

De acuerdo con la documentación oficial, Vue se destaca por tener una curva de aprendizaje accesible y por poder integrarse mediante un CDN sin necesidad de herramientas de compilación [1]. Estas características lo hacían adecuado para un proyecto académico con tiempo limitado.

¿Cómo se utilizó en este proyecto?

Vue.js se implementó en las cuatro ventanas del CRUD (create, read, update y delete) aplicando los siguientes conceptos:

  • Vinculación de datos con v-model en los formularios
  • Renderizado de listas con v-for en las tablas
  • Condicionales con v-if para vistas y modales
  • computed properties para el buscador en tiempo real
  • Manejo de eventos con @click y @submit
  • Hook mounted() para cargar datos al iniciar
